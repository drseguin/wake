# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

The date-based build identifier (e.g. `v2026.04.11.1430`) generated by
`./start.sh` is **not** a release version; it only tells you which binary is
running. The semantic version below (e.g. `1.0.0`) is the release version
and lives in `frontend/package.json`.

## [Unreleased]

### Added
- `USE_BASELINE_APP.md` with full clone/rename workflow and `scripts/rename-app.sh` helper (with auto-rotation of `FLASK_SECRET_KEY` and Keycloak client secret).
- `AuthContext` + `useAuth()` hook exposing `{ user, adminRole, isAdmin, hasRole }`.
- Backend `@require_auth` and `@require_role` decorators (`backend/utils/auth.py`).
- `ErrorBoundary`, `Protected` route wrapper, `NotFound` page.
- UI primitives: `Tooltip`, `FormField`, `EmptyState`.
- `POST /api/v1/auth/refresh` and single-flight 401→refresh retry in `services/api.js`.
- End-to-end `X-Request-ID` propagation (`g.request_id`, logger filter, response header, frontend log lines).
- Level-aware frontend logger with runtime `setLevel()`, synced from backend `LOG_LEVEL` via `/api/v1/config`.
- Docker healthchecks and `./start.sh` health polling (replaces `sleep 3`).
- `GET/PUT /api/v1/user/preferences` with `PreferencesSync` component; theme + accent persist per user.
- `./start.sh --env <name>` and `nginx/nginx.conf.template` for multi-env support.
- `scripts/install-hooks.sh` installing a pre-commit check for unified-logger usage and `@fileoverview` headers.
- Branding placeholders under `frontend/public/` (favicon.svg, apple-touch-icon.svg, og-image.svg) and Open Graph meta tags in `index.html`.

### Changed
- Header logo now renders from `appName` config instead of hardcoded "Base App" spans.
- `/api/v1/config` now includes `admin_role` and `log_level`.
- `rename-app.sh` extended to cover `frontend/src/**` and `frontend/public/**`.
- `CLAUDE.md` architectural docs refreshed; logger prefix consistently documented as `DSC:`.
- Triton API reference moved to `REF_APPS/Translate/documentation/`.

### Fixed
- Logger prefix docs in `CLAUDE.md`, `README.md`, and `USE_BASELINE_APP.md` (were `BA:`, now `DSC:`).
- `UserMenu.jsx` admin check no longer hardcodes `'base-app-admin'` — uses `useAuth().isAdmin`.

## [1.0.0] - 2026

### Added
- Initial baseline extracted from the Translate reference app.
- React 18 + Vite + single `App.css` shell with light/dark + accent colors.
- Flask + gunicorn backend with Keycloak OAuth2 + PKCE, Redis sessions.
- Nginx reverse proxy with self-signed TLS.
- `./start.sh` / `./stop.sh` Docker workflow.
